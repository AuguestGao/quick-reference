# VS Code Guide

## Installation

go on vs code website, download and install

### add 'code' to path: enable code in command line

in vs code, `ctrl + shift + p` to open command pallete
type in `install path`
now you can use `code folder_name` in the command line

## add ons

### prettier

install "prettier"

#### config to run prettier on save

1. in code > preference > setting
2. search for 'format on save', and check the box
